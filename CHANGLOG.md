### CHANGELOG.md
```markdown
# Changelog

All notable changes to Tinkerflow-AI will be documented in this file.

## [1.0.0] - 2025-03-15

### Added
- Deterministic governance engine with cryptographic proofs.
- Multi-provider LLM gateway (Ollama, OpenAI, Anthropic).
- GitHub App integration with check runs and PR comments.
- Pro tier with custom policies and audit logs.

### Changed
- Configuration format migrated from YAML to JSON.
- Environment variable namespace changed to `TF_`.

### Fixed
- Signature verification for webhooks.
- Rate limiting handling for GitHub API.

## [0.9.0] - 2025-02-01

### Added
- Initial beta release with secret scanning and commit message checks.
- Support for self-hosted Ollama.
```
