## ✨ Features
- **Visual Node Programming** – Drag, drop, and connect nodes to create Python scripts without writing boilerplate.
- **AI Tutor** – Ask the AI to explain any Python concept or code snippet in plain English.
- **Live Code Generation** – Automatically generate Python code from your node graph.
- **Beginner Friendly** – Perfect for learning loops, conditionals, functions, and more.
