# Frequently Asked Questions

### ❓ General Questions

**Q: Is Tinkerflow free to use?**  
A: Yes, the core tool is free and open‑source. Some advanced features (like unlimited AI queries) may require a support tier – see [SUPPORT.md](../SUPPORT.md).

**Q: Do I need to know Python to use it?**  
A: Not at all! Tinkerflow is designed to teach Python visually. The AI tutor will guide you.

**Q: Can I use Tinkerflow offline?**  
A: Yes, if you run it locally. The AI features require an internet connection unless you set up a local model.

---

### 🛠️ Technical Questions

**Q: What file format does Tinkerflow use to save graphs?**  
A: Graphs are saved as `.tflow` files (JSON format). You can open them with any text editor.

**Q: How do I report a bug?**  
A: Open an issue on [GitHub Issues](https://github.com/rickcreator87/Tinkerflow-AI/issues) and include steps to reproduce.

**Q: Can I contribute my own node types?**  
A: Absolutely! See [Contributing](contributing.md) for details on extending Tinkerflow.

---

### 🤖 AI Tutor Questions

**Q: Does the AI store my conversations?**  
A: No, conversations are ephemeral and not stored on our servers. If you're running locally, they stay on your machine.

**Q: Which AI model is used?**  
A: The online demo uses GPT‑3.5. For local use, you can configure it to use any OpenAI‑compatible API.

**Q: Why did the AI give a wrong answer?**  
A: AI can make mistakes. If you think the answer is incorrect, please report it so we can improve.

---

### 💰 Funding & Support

**Q: What do I get if I become a GitHub Sponsor?**  
A: Depending on the tier, you get priority support, feature voting, monthly roadmap previews, and more. See [SUPPORT.md](../SUPPORT.md).

**Q: How can I support without money?**  
A: Star the repo, share it with friends, contribute code or documentation, or help answer questions in Discussions.
