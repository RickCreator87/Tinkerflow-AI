# Privacy Policy for Tinkerflow-AI GitHub App

The Tinkerflow-AI app accesses only the data necessary to perform governance checks on pull requests. We do not store private repository data beyond temporary caching for proof verification.

- **Data Collected**: PR metadata, commit SHAs, file diffs (only during check run).
- **Data Retention**: Immediately deleted after check completion.
- **Third Parties**: No data shared with third parties.

For questions, contact privacy@tinkerflow.ai.
