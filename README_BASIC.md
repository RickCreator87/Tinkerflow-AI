# Tinkerflow-AI – Basic Tier

Tinkerflow-AI brings deterministic governance to your pull requests using AI-powered checks.

## Features

- **Secret Scanning** – Detects hardcoded secrets in diffs.
- **Commit Convention Enforcement** – Ensures commits follow Conventional Commits.
- **License Header Check** – Verifies new files include license headers.
- **Free for Public Repos** – Unlimited checks on public repositories.

## Installation

1. Go to [github.com/apps/tinkerflow-ai](https://github.com/apps/tinkerflow-ai).
2. Click "Install" and select the repositories you want to enable.
3. That's it! Open a PR to see governance in action.
