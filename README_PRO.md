# Tinkerflow-AI – Pro Tier

Tinkerflow-AI is a deterministic governance engine for GitHub pull requests, powered by LLMs. The Pro tier adds advanced features for enterprise teams.

## Pro Features

- **Advanced Policies** – Custom policy definitions using Rego or JavaScript.
- **Multi-Provider Support** – Route requests to OpenAI, Anthropic, or self-hosted Ollama.
- **Audit Logs** – Full traceability of all governance checks.
- **SLA Support** – 99.9% uptime guarantee with priority support.

## Getting Started

1. Install the GitHub App from the [marketplace](https://github.com/marketplace/tinkerflow-ai-pro).
2. Configure policies in your repository under `.github/tinkerflow/policies.yml`.
3. Trigger a check by opening a pull request.
