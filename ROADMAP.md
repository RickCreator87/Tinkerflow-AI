# Roadmap

## Q2 2026

- **Custom Policy Language**: Allow policies written in Rego or JavaScript.
- **GitHub Actions Integration**: Run governance as a GitHub Action.
- **Dashboard**: Web UI for viewing governance history.

## Q3 2026

- **Enterprise SSO**: Support SAML/OIDC authentication.
- **Compliance Reports**: Generate SOC2/ISO reports from governance logs.
- **Slack Integration**: Notifications for governance failures.

## Q4 2026

- **Automated Remediation**: AI-suggested fixes for failing policies.
- **On-Premise Support**: Air-gapped deployment option.
