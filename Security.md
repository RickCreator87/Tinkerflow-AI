# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Tinkerflow-AI, please report it via private email to **security@tinkerflow.ai**. Do not open public issues.

We will acknowledge receipt within 48 hours and provide a timeline for a fix. Security fixes are backported to the latest minor version.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |
